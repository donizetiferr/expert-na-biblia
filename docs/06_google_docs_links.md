## Acesso e Permissoes

- Os links estao em modo "qualquer pessoa com o link pode ver" (compartilhamento publico)
- Caso algum link retorne erro de acesso, o usuario precisa:
  1. Verificar se a conta Google do usuario ainda tem acesso ao Drive
  2. Reativar compartilhamento publico
  3. Ou compartilhar o link com a conta do desenvolvedor para acesso direto
